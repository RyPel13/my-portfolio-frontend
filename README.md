# Portfolio Frontend

A React single-page application that displays my personal developer portfolio. Fetches live data from a Spring Boot REST API and presents it in a dark mode UI with filterable cards for projects, interests, and goals.

---

## Tech Stack

- React 18
- JavaScript (ES6+)
- Axios
- Inline CSS / Inline Styles
- Create React App

---

## Getting Started

### Prerequisites
- Node.js 16+
- npm
- VS Code (recommended)

### Run Locally
1. Clone the repository
```bash
   git clone https://github.com/RyPel13/my-portfolio-frontend.git
```
2. Install dependencies
```bash
   cd my-portfolio-frontend
   npm install
```
3. Make sure the backend is running at `http://localhost:8080`
4. Start the development server
```bash
   npm start
```
5. Open `http://localhost:3000` in your browser

---

## Features

- **About** — Developer name, introduction, email, GitHub, LinkedIn and Resume links
- **Projects** — Filterable project cards by language (Java, Python, JavaScript, SQL) linking to GitHub repos
- **Interests** — Filterable interest cards by category (Technology, Other)
- **Goals** — Filterable goal cards by status (All, Achieved, Pending)
- **Navbar** — Sticky navigation with smooth scroll to each section
- **Dark Mode** — Full dark theme with cyan and red accents
